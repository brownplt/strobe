Sudoku Solutionator


Pushing the mouse down on a game cell produces a picker that displays a set of numbers. The number that is selected when the mouse button goes up will be input into the cell.

The game has two modes: Setup and play. Cells that are entered in setup mode are protected in play mode.

The solutionator uses a brute-force algorithm that is guaranteed to solve any puzzle that has a solution, but it is not very clever in how it goes about it. It can take a long time to find a solution, but it does not lock up the browser. This is a demonstration of eteration.
